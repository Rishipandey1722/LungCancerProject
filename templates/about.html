My Lung Cancer Prediction Project

Objective

My goal is to learn and build a machine learning model that predicts the likelihood of lung cancer based on various health and lifestyle factors of patients.

What I've Done

  • Data Collection: I collected a dataset with 26 attributes and 1000 instances.
  • Data Cleaning and Preprocessing: Cleaned the dataset to handle missing values and outliers, ensuring data quality. Preprocessed the data by encoding categorical variables, scaling numerical features, and splitting it into training and testing sets.
  • Exploratory Data Analysis (EDA): Explored the data to understand attribute distributions, correlations, and patterns related to lung cancer levels (high, medium, low).
  • Feature Selection: Identified important features using correlation analysis and feature importance scores from models. Chose relevant features that significantly impact predicting lung cancer levels.
  • Model Selection and Training: Selected and trained multiple classification algorithms suitable for categorical data, including Logistic Regression, Naive Bayes, and Random Forest.
  • Model Evaluation: Evaluated model performance using metrics such as accuracy, precision, recall, and F1-score. Conducted cross-validation to ensure models generalize well to new data.
  • Model Ensembling: Implemented a Voting Classifier to combine predictions from Logistic Regression, Naive Bayes, and Random Forest for improved accuracy and reliability.
  • Web Interface Development: Designed a user-friendly web interface using Flask, HTML, and CSS. Created pages for user input, prediction results, project details, process flow, and other informative sections.
  • Deployment: Deployed the machine learning model and web application on platforms like PythonAnywhere for public access. Ensured the application is functional and accessible to users for predicting lung cancer risk based on their input data.

Tools used:

  • Python
  • Flask 2.0.1
  • Scikit-learn 0.24.2
  • Numpy , Pandas
  • HTML5/CSS3 for front-end development
  • Google collab Notebook for initial data exploration and analysis

Future Plans

  • Tuning and Optimization: Plan to fine-tune models, optimize hyperparameters, and explore advanced techniques for better predictions.
  • User Engagement: Gather user feedback and update the application based on user needs.
  • Expansion: Consider adding more features, datasets, or predictive models related to respiratory health or other medical conditions.
